Welcome to the Temperature Sensor API

This is the homepage for your API. To interact with the endpoints, please use the following URLs:

HTTP Endpoints:

Submit Temperature Reading (POST):

/api/temperature_reading

Method: POST

Requires authentication via X-API-Key in the header.

Example Request Body (JSON):

{
    "temperature": 25.5,
    "sensor_id": "sensor_001"
}

WebSocket Endpoint:

Receive Real-time Sensor Updates:

/ws/sensor_updates

Protocol: WebSocket

Connects to receive temperature updates as they are sent.

Requires authentication via api-key as a query parameter in the URL, for example:

ws://your-domain.com/ws/sensor_updates?api-key=YOUR_SECRET_KEY
