Sobre este Projeto

Este projeto demonstra uma arquitetura de software para o monitoramento de dados em tempo real. A aplicação é responsável por se conectar aos sensores, coletar, processar e armazenar os dados de forma eficiente.

Os dados processados são então disponibilizados para a plataforma Grafana, onde os operadores podem criar dashboards poderosos e customizados para garantir a conformidade com os padrões de qualidade e otimizar a produção.
